# Neural-analysis
Code for analyzing neural calcium responses

1. runThis_function_github.m. This file is for the basic fluorescence analysis for the visual stimulation runs (our standard is 8 orientations)
2. runThis_functionSimpleNPCorr_github.m. This file is for the basic fluorescence analysis when there are no conditions to average (our optogenetic constriction runs)
3. plot_Ch2_subtr_github.m. This file is to do the correction - usually takes two basic runs and takes the ratio of them for the correction.
4. pop_ratio_corr_github.m. This file is run after all the individual correction runs to do population averages and summaries
